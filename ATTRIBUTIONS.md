# Attributions

This project leverages several amazing open-source libraries to deliver a premium AI experience. We are grateful to the following maintainers and contributors:

## Flutter Ecosystem

- **flutter_inappwebview**: For the robust browser engine and session management capabilities.
- **flutter_bloc**: For the predictable and scalable state management architecture.
- **shared_preferences**: For local preference storage.
- **path_provider**: For providing access to device file systems.
- **http**: For networking and API interactions.
- **cached_network_image**: For efficient image loading and caching.
- **glassmorphism**: For the stunning visual effects and translucent UI components.
- **uuid**: For unique identifier generation.
- **intl**: For internationalization support.
- **url_launcher**: For launching external browser windows and apps.
- **home_widget**: For the Android home screen widget integration.
- **permission_handler**: For managing device permissions.
- **flutter_local_notifications**: For system-level alerts and notifications.
- **timezone**: For accurate time and scheduling.
- **flutter_secure_storage**: For encrypted local storage.
- **encrypt**: For the dual-layer AES-256 encryption logic.
- **crypto**: For hashing and security utilities.
- **gal**: For gallery saving functionality.
- **flutter_custom_tabs**: For secure SSO login flows.

## Testing Tools

- **mocktail**: For the clean and intuitive mocking interface.
- **bloc_test**: For standardizing the BLoC testing patterns.

## Design Inspiration

- **Apple iOS**: For the glassmorphic aesthetics, animations, and premium feel.
- **Modern AI Platforms**: ChatGPT, Claude, Gemini, and others for their service APIs and web interfaces.
